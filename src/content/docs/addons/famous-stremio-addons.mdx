## Popular Catalog Add-ons

### Torrentio

**Description**: One of the most popular torrent-based streaming add-ons.

**Features**:
- High-quality torrent sources
- Multiple indexers support (Jackett, Prowlarr)
- Debrid service integration
- Customizable quality preferences

**Installation**: Search for "Torrentio" in Stremio's add-on catalog.

**Status**: Check [status.dinsden.top](https://status.dinsden.top/status/stremio-addons) for current availability.

### Stremio Enhanced

**Description**: Enhanced version of official catalogs with additional features.

**Features**:
- Improved metadata
- Additional streaming sources
- Better poster quality
- Enhanced search capabilities

**Installation**: Search for "Stremio Enhanced" in the add-on catalog.

### Jackett

**Description**: Meta-search engine for torrent trackers.

**Features**:
- Supports 100+ torrent trackers
- Indexer aggregation
- API for other applications
- Docker support

**Installation**: Requires separate Jackett server setup. Visit [github.com/Jackett/Jackett](https://github.com/Jackett/Jackett) for installation guide.

## Popular Metadata Add-ons

### IMDb

**Description**: Official IMDb integration for ratings and information.

**Features**:
- Official IMDb ratings
- Cast and crew information
- User reviews and scores
- Comprehensive movie data

**Installation**: Built into Stremio by default.

### Trakt.tv

**Description**: Social media platform for tracking watched media.

**Features**:
- Watch history tracking
- Personalized recommendations
- Lists and collections
- Cross-device sync

**Installation**: Search for "Trakt" in the add-on catalog.

### Rating Poster Database (RPDB)

**Description**: High-quality posters and artwork database.

**Features**:
- 4K poster support
- Multiple poster options
- Custom artwork collections
- Community contributions

**Installation**: Visit [ratingposterdb.com](https://ratingposterdb.com/) for setup instructions.

## Subtitle Add-ons

## Installation Steps

<Steps>
1. **Check Status**: Visit [status.dinsden.top](https://status.dinsden.top/status/stremio-addons) to verify add-on availability
2. **Open Add-on Browser**: In Stremio, go to the add-on section
3. **Search**: Use the search function to find the desired add-on
4. **Install**: Click install and follow any setup prompts
5. **Configure**: Some add-ons require additional configuration (API keys, server setup)
6. **Test**: Try streaming content to ensure it works properly
</Steps>

## Troubleshooting

### Add-on Not Working

<Steps>
1. **Check Status**: Verify add-on is online at [status.dinsden.top](https://status.dinsden.top/status/stremio-addons)
2. **Restart Stremio**: Close and reopen the application
3. **Reinstall Add-on**: Remove and reinstall the add-on
4. **Check Network**: Ensure stable internet connection
5. **Clear Cache**: Clear Stremio's cache in settings
</Steps>

### Common Issues

- **Add-on shows as offline**: Check status page for maintenance
- **No streams available**: Add-on may be rate-limited or blocked
- **Poor quality**: Adjust quality settings in add-on configuration
