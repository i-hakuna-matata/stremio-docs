## What is AIOStream?

**AIOStream** is a popular Stremio add-on that aggregates streaming sources from multiple providers to find available streams for movies and TV shows. It works by:

- **Scraping torrent sites** and indexing services
- **Connecting to file hosting services** (Mega, MediaFire, etc.)
- **Integrating with debrid services** for premium access
- **Aggregating multiple stream sources** into a unified interface

The add-on provides a comprehensive stream-finding solution that searches across various platforms simultaneously.

## Installation

### Official Installation Links

Install AIOStream from one of these verified sources:

- **GitHub Repository**: [https://github.com/Viren070/AIOStreams](https://github.com/Viren070/AIOStreams)
- **Stremio Add-on Catalog**: Search for "AIOStream" in the Community Add-ons section
- **Direct Install URL**: `https://aiostreams.github.io/install.html`

### Installation Steps

<Steps>
1. **Open Stremio** and click the **puzzle icon** (Add-ons) in the left sidebar
2. **Click "Community Add-ons"** at the top
3. **Search for "AIOStream"** in the search bar
4. **Click "Install"** on the AIOStream add-on
5. **Wait for installation** to complete (may take a few seconds)
6. **Restart Stremio** if prompted
</Steps>

### Alternative Installation (Manual)

If the automatic installation doesn't work:

<Steps>
1. Copy this URL: `https://aiostreams.github.io/manifest.json`
2. In Stremio, go to **Add-ons → Community Add-ons**
3. Click **"Install from URL"**
4. Paste the manifest URL and click **Install**
</Steps>

## How AIOStream Works

### Stream Sources

AIOStream aggregates streams from multiple sources:

- **Torrent streams** (via torrent indexing sites)
- **Direct file links** (Mega, MediaFire, Google Drive, etc.)
- **Debrid service integration** (Real-Debrid, Premiumize, AllDebrid)
- **Web player streams** (various hosting platforms)

### Stream Quality Options

The add-on provides multiple quality options:
- **4K UHD** (when available)
- **1080p HD**
- **720p HD**
- **480p SD**
- **Various audio languages** and subtitle options

### Search and Discovery

- **Real-time search** across all configured sources
- **Automatic quality ranking** (higher quality streams prioritized)
- **Multiple source fallback** (if one source fails, others are tried)
- **Caching** for faster subsequent searches

## Configuration

### Basic Setup

After installation, AIOStream works with default settings, but you can optimize it:

<Steps>
1. **Open Add-ons** in Stremio
2. **Click the gear icon** next to AIOStream
3. **Configure basic settings** (see sections below)
4. **Test with a popular movie/TV show**
</Steps>

### Stream Source Configuration

#### Torrent Sources

Configure which torrent sites to search:

```json
{
  "torrentSources": [
    "1337x",
    "ThePirateBay",
    "RARBG",
    "Torrentz2",
    "YTS"
  ]
}
```

#### File Hosting Services

Enable/disable specific hosting services:

```json
{
  "fileHosts": {
    "mega": true,
    "mediafire": true,
    "googledrive": true,
    "dropbox": false,
    "onedrive": true
  }
}
```

### Debrid Service Integration

For premium users with debrid services:

<Steps>
1. **Get your API key** from your debrid service provider
2. **In AIOStream settings**, go to **Debrid** section
3. **Select your provider** (Real-Debrid, Premiumize, AllDebrid)
4. **Enter your API key**
5. **Test the connection**
</Steps>

**Supported Debrid Services:**
- Real-Debrid
- Premiumize
- AllDebrid
- Debrid-Link

### Quality Preferences

Configure quality and format preferences:

```json
{
  "preferredQuality": ["4K", "1080p", "720p"],
  "preferredFormats": ["MP4", "MKV", "AVI"],
  "maxResults": 20,
  "timeout": 10000
}
```

### Advanced Settings

#### Caching Configuration

```json
{
  "cache": {
    "enabled": true,
    "ttl": 3600000,  // 1 hour in milliseconds
    "maxSize": 100   // Maximum cached results
  }
}
```

#### Network Settings

```json
{
  "network": {
    "timeout": 15000,
    "retries": 3,
    "userAgent": "Stremio-AIOStream/1.0"
  }
}
```

## Troubleshooting

### Common Issues

#### No Streams Found

**Possible causes:**
- Torrent sites are down or blocked
- Your IP is rate-limited
- VPN/proxy interfering

**Solutions:**
- Try different VPN server
- Wait a few minutes and retry
- Check if torrent sites are accessible

#### Slow Loading

**Optimization tips:**
- Reduce the number of active sources
- Increase timeout values
- Enable caching
- Use a faster VPN server

#### Playback Issues

**For stream playback problems:**
- Try different quality options
- Switch to a different source
- Check your internet connection
- Clear Stremio cache

### Debug Mode

Enable debug logging for troubleshooting:

<Steps>
1. **Open AIOStream settings**
2. **Go to Advanced section**
3. **Enable "Debug Mode"**
4. **Check Stremio console** for error messages
5. **Share logs** when reporting issues
</Steps>

## Updates and Maintenance

### Checking for Updates

AIOStream updates automatically, but you can manually check:

<Steps>
1. **Go to Add-ons → Installed**
2. **Find AIOStream**
3. **Click "Check for Updates"**
4. **Install updates** if available
</Steps>

### Clearing Cache

If you experience issues, clear the add-on cache:

<Steps>
1. **Open AIOStream settings**
2. **Click "Clear Cache"**
3. **Restart Stremio**
</Steps>

## Performance Optimization

### Best Practices

1. **Use a VPN** for privacy and access
2. **Configure only needed sources** to reduce search time
3. **Enable caching** for frequently watched content
4. **Set reasonable timeouts** (10-15 seconds)
5. **Use debrid services** for faster, more reliable streams

### Recommended Settings

For optimal performance:

```json
{
  "quality": ["1080p", "720p"],
  "maxResults": 15,
  "timeout": 12000,
  "cache": true,
  "debrid": true
}
```
