Stremio's sync feature allows you to access your library, watchlist, and viewing progress across all your devices. This guide covers setting up sync and managing multiple devices effectively.

## Setting Up Sync

### Creating a Stremio Account

<Steps>

1. **Sign Up Process**
   - Open Stremio app
   - Click profile icon (top right)
   - Select "Login" or "Create Account"
   - Enter email and password

2. **Email Verification**
   - Check email for verification link
   - Click link to activate account
   - Return to app and login

3. **Account Benefits**
   - Sync across unlimited devices
   - Backup of your data
   - Access to premium features
   - Community features

</Steps>

<Callout type="info">
Stremio accounts are free and don't require payment information. Premium features are optional.
</Callout>

### Initial Sync Setup

<Steps>

1. **Login on First Device**
   - Enter credentials
   - Allow data synchronization
   - Initial sync may take a few minutes

2. **Verify Sync Status**
   - Check profile icon shows logged in status
   - Library and watchlist should appear
   - Settings sync across devices

3. **Add Additional Devices**
   - Install Stremio on other devices
   - Login with same account
   - Data automatically syncs

</Steps>

## What Gets Synced

### Core Data

**Always Synced:**
- **Library**: Your personal collection
- **Watchlist**: Saved for later content
- **History**: Viewing history and progress
- **Add-ons**: Installed add-ons and settings
- **User Preferences**: App settings and configurations

**Device-Specific:**
- **Local Files**: Device-specific media
- **Cache**: Temporary data (not synced)
- **Offline Content**: Downloaded content

### Sync Timing

**How Often It Syncs:**
- **Real-time**: Changes sync within seconds
- **On App Start**: Full sync when opening app
- **Manual Sync**: Force sync in settings
- **Background**: Continuous sync when connected

<Callout type="tip">
Sync works over Wi-Fi and mobile data. Enable background sync for best experience.
</Callout>

### Device Management

<Steps>

1. **View Connected Devices**
   - Settings → Account
   - See list of active devices
   - Last sync time for each device

2. **Device Limits**
   - Unlimited devices allowed
   - No restrictions on device types
   - Simultaneous usage supported

3. **Remove Devices**
   - Select device from list
   - Choose "Remove" or "Logout"
   - Device loses access to account

</Steps>

### Device-Specific Settings

## Sync Troubleshooting

### Common Sync Issues

**Not Syncing:**
- Check internet connection
- Verify login status
- Restart Stremio app
- Check account status

**Missing Data:**
- Wait for sync to complete
- Check other devices for data
- Force manual sync
- Verify account ownership

**Slow Sync:**
- Large library may take time
- Check connection speed
- Close other apps using bandwidth
- Sync during off-peak hours

### Advanced Troubleshooting

<Steps>

1. **Clear Sync Cache**
   - Settings → Advanced
   - Clear sync data
   - Re-login to account

2. **Reset Sync**
   - Logout from all devices
   - Clear app data/cache
   - Re-login and allow fresh sync

3. **Network Issues**
   - Check firewall settings
   - Try different network
   - Disable VPN temporarily
   - Contact ISP if needed

</Steps>

## Privacy & Security

### Data Privacy

**What Stremio Stores:**
- Your library and watchlist
- Viewing progress and history
- Add-on preferences
- Anonymous usage analytics

**What Stremio Doesn't Share:**
- Specific content you watch (only aggregated)
- Personal information with third parties
- Viewing data for advertising

### Account Security

<Steps>

1. **Strong Password**
   - Use complex, unique password
   - Enable two-factor authentication (2FA)
   - Change password regularly

2. **Device Management**
   - Regularly review connected devices
   - Remove unknown or unused devices
   - Logout from public/shared devices

3. **Privacy Settings**
   - Control analytics sharing
   - Manage data retention
   - Export/delete account data

</Steps>

<Callout type="warning">
Always logout from shared or public devices. Use strong passwords and enable 2FA for security.
</Callout>

## Best Practices

✅ **Do:**
- Use strong, unique passwords
- Enable 2FA for security
- Regularly review connected devices
- Keep apps updated for sync improvements
- Backup important data manually
- Use same account across personal devices
- Logout from shared/public devices

❌ **Don't:**
- Share account credentials
- Use sync on untrusted networks without VPN
- Ignore sync error messages
- Clear data without backup
- Use multiple accounts unnecessarily
- Disable sync for convenience
