## General Troubleshooting Steps

Before diving into specific guides, try these universal fixes:

### 1. Restart Stremio

- **Close** Stremio completely (don't just minimize)
- **Wait** 10 seconds
- **Reopen** Stremio

**Fixes:** Many transient issues, memory leaks, temporary glitches.

### 2. Check Your Internet

- **Test speed:** Go to [fast.com](https://fast.com) or [speedtest.net](https://www.speedtest.net)
- **Required:** 5 Mbps for SD, 25 Mbps for HD, 50+ Mbps for 4K
- **Restart your router** if speed is slow

**Fixes:** Buffering, "No connection" errors, slow loading.

### 3. Clear Cache

- Go to **Settings** ‚Üí **Advanced**
- Click **"Clear Cache"**
- Restart Stremio

**Fixes:** Playback errors, corrupted data, interface glitches.

### 4. Update Stremio

- Check if an update is available (Stremio notifies you)
- **Download** the latest version from [stremio.com](https://www.stremio.com)
- **Install** over your existing version

**Fixes:** Bugs fixed in newer versions, compatibility issues.

### 5. Check Firewall/Antivirus

- Ensure Stremio is **allowed** through your firewall
- **Temporarily disable** antivirus and test (if safe to do so)
- **Add Stremio to exceptions** if it was being blocked

**Fixes:** "Connection refused," streaming failures.

## Problem Categories Explained

### Playback Issues

**Symptoms:**
- Video won't start
- "No streams available"
- Infinite buffering
- Stuttering or freezing during playback

**Common causes:**
- Slow or unstable internet
- Add-on issues (broken or offline)
- Firewall blocking Stremio
- Insufficient bandwidth

**Guide:** [Playback & Buffering](/troubleshooting/playback-buffering/)

---

### Subtitle Issues

**Symptoms:**
- Subtitles don't appear
- Wrong language
- Out of sync with audio
- Subtitle add-on not working

**Common causes:**
- No subtitle add-on installed
- Subtitles not available for that title
- Language mismatch
- Timing issues

**Guide:** [Subtitles Not Working](/troubleshooting/subtitles-not-working/)

---

### Audio/Video Sync Issues

**Symptoms:**
- Audio plays before or after video
- Audio track missing
- Wrong language audio

**Common causes:**
- Stream quality issues
- Device performance problems
- Player settings

**Guide:** [Audio/Video Sync](/troubleshooting/audio-video-sync/)

---

### App Stability Issues

**Symptoms:**
- Stremio won't launch
- Crashes on startup
- Freezes during use
- "Not responding" errors

**Common causes:**
- Corrupted installation
- Insufficient device resources
- Conflicting software
- OS compatibility issues

**Guide:** [App Crashes & Freezes](/troubleshooting/app-crashes-and-freezes/)

---

### Login/Sync Issues

**Symptoms:**
- Can't log in
- "Invalid credentials" error
- Library not syncing across devices
- Account settings not saving

**Common causes:**
- Wrong email/password
- Network issues
- Server problems (Stremio's side)
- Cache corruption

**Guide:** [Login & Sync Issues](/troubleshooting/login-sync-issues/)

### Android TV Performance

**Symptoms:**
- Slow, laggy interface
- App crashes frequently on TV
- Remote control unresponsive

**Common causes:**

- Too many apps running
- Insufficient storage

## When to Reinstall

Reinstalling should be a **last resort**, but it's necessary if:

- Stremio won't launch at all after trying everything
- Corruption is suspected (frequent crashes, data loss)
- You've exhausted all other troubleshooting steps

### How to Reinstall Cleanly

1. **Uninstall Stremio** (see your device-specific install guide)
2. **Delete user data folder:**
   - **Windows:** `%APPDATA%\Stremio`
   - **macOS:** `~/Library/Application Support/Stremio`
   - **Linux:** `~/.config/Stremio`
   - **Android:** Clear app data in Settings ‚Üí Apps ‚Üí Stremio
3. **Restart your device**
4. **Download fresh installer** from [stremio.com](https://www.stremio.com)
5. **Install** and set up from scratch

<Callout type="warning">
**Warning:** Deleting user data erases local settings and cache. Your library is safe if you have a Stremio account (cloud-synced).
</Callout>

## Prevention Tips

Avoid future issues with these best practices:

### ‚úÖ DO:

- **Keep Stremio updated** (install updates when notified)
- **Use a stable internet connection** (wired Ethernet is best)
- **Audit add-ons regularly** (remove broken or unused ones)
- **Clear cache periodically** (once a month if you use Stremio daily)
- **Restart Stremio occasionally** (don't leave it running 24/7)

### ‚ùå DON'T:

- **Install too many add-ons** (more add-ons = more complexity)
- **Ignore error messages** (they often tell you what's wrong)
- **Modify advanced settings** without understanding them
- **Use Stremio on an overloaded/underpowered device**
