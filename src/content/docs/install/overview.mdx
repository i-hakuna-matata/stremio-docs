## Before You Install

### System Requirements

**Minimum:**
- **Processor:** Dual-core CPU (2 GHz)
- **RAM:** 2 GB
- **Storage:** 200 MB for app, plus cache (1-5 GB recommended)
- **Internet:** 5 Mbps for SD, 25 Mbps for HD, 50+ Mbps for 4K
- **OS:** See device-specific guides above

**Recommended:**
- Quad-core CPU or better
- 4 GB+ RAM
- SSD storage for better performance
- Stable, high-speed internet connection

<Callout type="note">
Stremio is lightweight compared to many media apps, but performance depends heavily on your internet speed and the add-ons you use.
</Callout>

### What You'll Need

For all installations:

- ‚úÖ Internet connection
- ‚úÖ 10-15 minutes of time
- ‚úÖ Administrator/root access (for desktop OS)
- ‚úÖ Ability to download files from the internet

For specific platforms:

- **Android (non-Google Play):** Ability to enable "Unknown Sources"
- **Linux:** Basic terminal knowledge (for some methods)
- **macOS:** May need to allow apps from "unidentified developers"

## Platforms NOT Supported

<Callout type="warning">
Stremio does NOT officially support the following platforms. Workarounds may exist but are not covered in this guide.
</Callout>

### ‚ùå iOS (iPhone/iPad)

**Why not:** Apple's App Store policies restrict apps like Stremio. There's no official iOS app.

**Workarounds:** 
- Use the web version (app.strem.io) in Safari‚Äîlimited functionality
- Use third-party apps that integrate with Stremio (not officially supported)
- Install Stremio on a different device and cast to Apple TV

### ‚ùå Smart TVs (Samsung, LG, Roku, etc.)

**Why not:** These platforms use proprietary operating systems. Stremio doesn't have native apps for them.

**Workarounds:**
- Use an **Android TV box** or **Fire TV Stick** (see [Android TV guide](/install/android-tv/))
- Cast from your phone or computer to the TV
- Use an HDMI cable to mirror your computer screen

### ‚ùå Game Consoles (PlayStation, Xbox, Nintendo)

**Why not:** Stremio doesn't have console versions.

**Workarounds:**
- Cast from another device
- Use the console's web browser (if available) to access app.strem.io‚Äîhighly limited

### ‚ùå Older Operating Systems

- **Windows:** XP, Vista (no longer supported)
- **macOS:** 10.10 Yosemite and earlier
- **Android:** Below 5.0 Lollipop

If you're on an older OS, you may need to upgrade or use a different device.

## Which Version Should I Install?

### Desktop (Windows, macOS, Linux)

**Recommendation:** Install the **latest stable release** from the official Stremio website.

- ‚úÖ Full features
- ‚úÖ Best performance
- ‚úÖ Regular updates

**Avoid:**
- ‚ùå Beta or development versions (unless you're an advanced user testing features)
- ‚ùå Unofficial third-party builds (security risk)

### Mobile (Android)

**Recommendation:** Install from **Google Play Store** if possible.

- ‚úÖ Automatic updates
- ‚úÖ Verified by Google
- ‚úÖ Easiest to keep secure

**If Google Play isn't available (e.g., Fire tablets):**
- Download the official APK from stremio.com
- Verify the file signature (advanced)

### Android TV

**Recommendation:** Install the **dedicated Android TV version** from Google Play or the official APK.

- ‚úÖ TV-optimized interface
- ‚úÖ Remote control support
- ‚úÖ Better performance on TV hardware

**Don't install:**
- ‚ùå The phone/tablet version on your TV (it'll work but the UI is not optimized)

## Security Best Practices

Before and during installation:

### ‚úÖ DO:

- **Download only from official sources** (stremio.com or official app stores)
- **Verify file checksums** if you're technically inclined
- **Keep your OS updated** before installing
- **Use antivirus/antimalware software** (especially on Windows)
- **Read permission requests** before granting access

### ‚ùå DON'T:

- **Download from third-party sites** (risk of malware)
- **Disable your firewall or antivirus** to install (if required, something is wrong)
- **Install cracked or "modified" versions** (huge security risk)
- **Skip reading what the installer asks for** (permissions, install location, etc.)

<Callout type="legal">
Only install Stremio from official sources. Third-party versions may contain malware, spyware, or unwanted modifications.
</Callout>

## Common Installation Questions

### Do I need an account to install?

**No.** You can install and use Stremio without an account. However, creating an account (free) is recommended for syncing across devices.

### How much does it cost?

**Stremio is free.** No purchase, no subscription, no hidden fees. Some add-ons may require paid services, but the app itself is free.

### Can I install on multiple devices?

**Yes!** Install on as many devices as you want. Create a Stremio account to sync your library across them.

### Is it legal to install Stremio?

**Yes.** Installing Stremio is 100% legal worldwide. What you do with it (the add-ons you install and content you access) determines legality. See [Safety & Legality](/start-here/safety-and-legality/).

### Will it slow down my device?

**Unlikely.** Stremio is lightweight. Performance depends more on your internet speed and the add-ons you use than on the app itself.
