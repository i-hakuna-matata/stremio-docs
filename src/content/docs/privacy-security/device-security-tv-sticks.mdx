Android TV sticks and streaming devices require special security considerations. This guide helps you secure your streaming devices and protect your privacy.

## Why TV Stick Security Matters

TV sticks and streaming boxes:
- Are always connected to the internet
- Often lack security updates
- May come pre-loaded with malware
- Can be entry points to your home network
- Store your account credentials

<Callout type="warning">
Many cheap Android TV boxes come with pre-installed malware, keyloggers, or backdoors. Always buy from reputable brands and sellers.
</Callout>

## Initial Setup Security

### First-Time Setup

<Steps>

1. **Factory Reset** (even if new)
   - Settings ‚Üí Device Preferences ‚Üí Reset
   - This removes potential pre-installed malware

2. **Update Immediately**
   - Settings ‚Üí System ‚Üí System Update
   - Install all available updates

3. **Disable Unknown Sources** (initially)
   - Settings ‚Üí Security & Restrictions
   - Turn off "Unknown Sources"
   - Enable only when installing trusted apps

4. **Create Strong PIN**
   - Settings ‚Üí Security & Restrictions ‚Üí PIN
   - Set a 4-6 digit PIN
   - Required for purchases and settings changes

</Steps>

### Secure Your Google Account

<Steps>

1. **Use a dedicated email** (optional but recommended)
   - Create a separate Google account for TV
   - Don't use your primary email
   - Easier to manage if compromised

2. **Enable 2FA** on your Google account
   - Visit [google.com/2fa](https://www.google.com/landing/2step/)
   - Secure the account linked to your TV

3. **Review app permissions**
   - Google Account ‚Üí Security ‚Üí Third-party apps
   - Remove unused or suspicious apps

</Steps>

## Installing Stremio Safely

### Official Installation

<Steps>

1. **Enable Unknown Sources** temporarily:
   - Settings ‚Üí Security & Restrictions ‚Üí Unknown Sources
   - Enable for browser or file manager

2. **Download from official source**:
   - Visit [stremio.com](https://www.stremio.com/downloads) on your TV browser
   - Download the Android TV APK
   - Or use **Downloader app** with official URL

3. **Install and Verify**:
   - Open the downloaded APK
   - Install Stremio
   - Launch and verify it's the official app

4. **Disable Unknown Sources** again:
   - Settings ‚Üí Security & Restrictions
   - Turn off Unknown Sources

</Steps>

<Callout type="tip">
For Fire TV, you can also use the Amazon Appstore if Stremio is available there, or sideload using the Downloader app with the official Stremio URL.
</Callout>

## Network Security

### Secure Your Wi-Fi

<Steps>

1. **Use strong Wi-Fi password**
   - WPA3 or WPA2 encryption
   - Long, complex password
   - Change default router password

2. **Guest Network** (recommended)
   - Create a separate guest network for TV devices
   - Isolates them from main network
   - Protects computers and phones

3. **Router Updates**
   - Keep router firmware updated
   - Check manufacturer website
   - Enable auto-updates if available

</Steps>

### Using a VPN

VPNs can enhance privacy on TV sticks:

**VPN Benefits:**
- Encrypts traffic
- Hides IP address
- Protects on public networks
- May bypass regional restrictions

**Recommended VPN Apps for Android TV:**
- ExpressVPN
- NordVPN
- Surfshark
- ProtonVPN
- Mullvad

<Callout type="info">
Configure VPN before opening Stremio for maximum privacy. Some free VPNs may compromise security‚Äîuse reputable paid services.

## App Security

### Safe App Installation

<Steps>

1. **Prefer Google Play Store**
   - Install apps from Play Store when possible
   - Verified and generally safer

2. **Verify Sources for Sideloaded Apps**
   - Only download APKs from official websites
   - Check file integrity if possible
   - Avoid third-party app stores

3. **Review Permissions**
   - Check what access apps request
   - Deny unnecessary permissions
   - Settings ‚Üí Apps ‚Üí Permissions

4. **Keep Apps Updated**
   - Settings ‚Üí Apps ‚Üí See all apps
   - Update regularly
   - Enable auto-updates in Play Store

</Steps>

### Avoid Malicious Apps

**Red flags:**
- Apps asking for excessive permissions
- Unknown developers
- Apps that can't be found in official stores
- Bundled with "free content" promises

## Physical Security

### Protect the Device

- **Secure Location**: Keep TV stick away from public access
- **HDMI Port Lock**: For public spaces, consider physical locks
- **Regular Resets**: Factory reset if selling or giving away

### Remote Control Security

- **PIN Protect Purchases**: Require PIN for all purchases
- **Disable Voice Recording**: Turn off microphone when not needed
- **Review Alexa/Google History**: Clear voice recordings regularly

## Privacy Settings

### Android TV Privacy

<Steps>

1. **Limit Data Collection**:
   - Settings ‚Üí Privacy
   - Disable usage statistics
   - Limit ad personalization

2. **Disable Suggestions**:
   - Settings ‚Üí Device Preferences ‚Üí Home Screen
   - Turn off personalized recommendations
   - Reduces data tracking

3. **Review Location Access**:
   - Settings ‚Üí Location
   - Turn off unless needed
   - Deny for apps that don't need it

4. **Microphone Privacy**:
   - Settings ‚Üí Privacy ‚Üí Microphone
   - Disable for apps that don't need it
   - Consider physical mic mute if available

</Steps>

## Regular Maintenance

### Weekly Tasks

<Steps>

1. **Check for Updates**
   - System updates
   - App updates
   - Security patches

2. **Review Installed Apps**
   - Remove unused apps
   - Check for suspicious apps
   - Verify all apps are needed

</Steps>

### Monthly Tasks

<Steps>

1. **Clear Cache**
   - Settings ‚Üí Apps ‚Üí Clear Cache
   - Improves performance
   - Removes temporary data

2. **Review Permissions**
   - Check app permissions
   - Revoke unnecessary access

3. **Check Network Activity**
   - Monitor unusual network usage
   - Check router logs if possible

</Steps>

### Quarterly Tasks

<Steps>

1. **Factory Reset** (optional)
   - Complete fresh start
   - Removes accumulated junk
   - Backup important data first

2. **Review Google Account**
   - Check linked devices
   - Remove old devices
   - Update passwords

</Steps>

## Signs of Compromise

### Warning Signs

üö© **Watch for:**
- Unexpected ads or pop-ups
- Apps you didn't install
- Slow performance
- Excessive data usage
- Device heating up unusually
- Random reboots
- Unknown network connections
- Weird behavior in apps

### If Compromised

<Steps>

1. **Disconnect from Internet**
   - Turn off Wi-Fi immediately

2. **Factory Reset**
   - Settings ‚Üí Device Preferences ‚Üí Reset
   - Wipe all data

3. **Change Passwords**
   - Google account password
   - Stremio account password
   - Any other linked accounts

4. **Scan Network**
   - Check other devices
   - Update router password
   - Consider router reset

5. **Report Device** (if purchased online)
   - Contact seller
   - Leave review warning others
   - Report to platform (Amazon, eBay, etc.)

</Steps>
